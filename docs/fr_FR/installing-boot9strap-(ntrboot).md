# Installation de boot9strap (ntrboot)

## Lecture requise

Pour utiliser les liens [magnet](https://fr.wikipedia.org/wiki/Magnet_(standard)) sur cette page, vous aurez besoin d'un client torrent comme [qBittorrent](https://www.qbittorrent.org/download.php) ou [Deluge](http://dev.deluge-torrent.org/wiki/Download).

## Ce dont vous avez besoin

- Un aimant qui déclenche le mode veille de votre console (si vous avez une console pliable)
- Votre linker flashé avec ntrboot
- La dernière version de [SafeB9SInstaller](https://github.com/d0k3/SafeB9SInstaller/releases/download/v0.0.7/SafeB9SInstaller-20170605-122940.zip) (téléchargement direct)
- La dernière version de [boot9strap](https://github.com/SciresM/boot9strap/releases/download/1.4/boot9strap-1.4.zip) (téléchargement direct)
- La dernière version de [Luma3DS](https://github.com/LumaTeam/Luma3DS/releases/latest) (le fichier \`.zip' de Luma3DS)

::: warning

If you are using a **DSpico** or an **Ace3DS X**, ensure that you have a blank microSD card inserted into its microSD card slot. You do not need to put any files onto this microSD card; these cartridges require a microSD card to be inserted in order to trigger ntrboot.

:::

## Instructions

### Section I - Préparatifs

1. Éteignez votre console

2. Insérez la carte SD de votre console dans votre ordinateur
   - Ceci est la carte SD de votre 3DS, _pas_ la carte SD de votre linker

3. Copiez `SafeB9SInstaller.firm` depuis l'archive SafeB9SInstaller `.zip` vers la racine de votre carte SD et renommez-le en `boot.firm`
   - La racine de la carte SD représente le répertoire initial sur votre carte SD où vous pouvez voir le dossier Nintendo 3DS, mais n'êtes pas à l'intérieur de celui-ci

4. Créez un dossier nommé `boot9strap` à la racine de la carte SD de votre console

5. Copiez `boot9strap.firm` et `boot9strap.firm.sha` depuis l'archive boot9strap `.zip` vers le dossier `/boot9strap/` sur la carte SD de votre console

   ::: info

   ![](/images/screenshots/boot9strap-ntrboot-file-layout.png)

   :::

6. Réinsérez votre carte SD dans votre console

7. Allumez votre console

### Section II - ntrboot

1. Utilisez l’aimant pour repérer l’emplacement sur votre console où le capteur de veille est déclenché
   - Cette étape n’est pas nécessaire sur les old 2DS (qui ont un interrupteur de mode veille)
2. Éteignez votre console
3. Insérez votre linker dans votre console
4. Place the magnet near the (A)(B)(X)(Y) buttons on your console to trigger the sleep sensor
   - Sur une Old 2DS, vous devriez à la place activer l'interrupteur de mode veille
5. Maintenez les boutons (Start) + (Select) + (X) + (Power) pendant plusieurs secondes, puis relâchez-les
   - Cela peut demander plusieurs essais pour que cela fonctionne car le positionnement des boutons est peu commode
6. Si l'exploit a réussi, vous aurez démarré sur SafeB9SInstaller
7. Retirez l'aimant de votre console
   - Sur une Old 2DS, vous devriez à la place désactiver l'interrupteur de mode veille

### Section III - Installation de boot9strap

Dans cette section, vous allez installer un custom firmware sur votre console.

1. Lorsque vous y êtes invité, entrez la combinaison de boutons donnée sur l'écran du haut pour installer boot9strap
   - Dans le cas où un message en rouge apparaîtrait lors d'une étape sur l'écran du bas et que vous n'êtes pas sollicité pour saisir une combinaison de boutons, [suivez ce guide de dépannage](troubleshooting-ntrboot)
2. Une fois terminé, forcez votre console à s'éteindre en maintenant le bouton Power enfoncé
   - Votre console ne démarrera que sur l'écran de SafeB9SInstaller jusqu'à ce que la section suivante soit complétée

### Section IV - Configuration de Luma3DS

1. Insérez la carte SD de votre console dans votre ordinateur
2. Supprimez le fichier `boot.firm` de la racine de votre carte SD
3. Copiez tout depuis le `.zip` de Luma3DS à la racine de votre carte SD
4. Réinsérez votre carte SD dans votre console
5. Allumez votre console

::: warning

**DSpico users** do not need to follow this section, as the hybrid firmware works for both ntrboot and normal flashcart use.

Notez que l’Acekard 2i conserve sa capacité de lancer des fichiers « .nds » tout en ayant l'exploit ntrboot installé. Cela s’applique uniquement lorsque l’Acekard 2i est dans une 3DS avec un custom firmware ou une NDS ! Tant que ntrboot est installé sur l’Acekard 2i, il lui sera impossible de lancer des fichiers `.nds` sur une 3DS non modifiée ou une DSi.

Ne suivez pas cette section tant que vous n'avez pas fini le reste des instructions de cette page.

### Section V (optional) - Removing ntrboot

#### Ce dont vous avez besoin

- La dernière version de [ntrboot_flasher](https://github.com/ntrteam/ntrboot_flasher/releases/latest) (`ntrboot_flasher.firm`)
- La sauvegarde de la flashrom correspondant à votre linker
  - Notez que si vous avez suivi [Flasher ntrboot (plusieurs 3DS)](flashing-ntrboot-(3ds-multi-system)), la sauvegarde de la flashrom existe déjà à l'emplacement correct et n'a pas besoin d'être téléchargée
  - Notez que si vous avez suivi [Flasher ntrboot (Une seule 3DS)](flashing-ntrboot-(3ds-single-system)) ou [Flash de ntrboot (NDS)](flashing-ntrboot-(nds)), la sauvegarde de la flashrom existe déjà sur la carte SD de votre linker et devrait être copiée à l'emplacement précisé ci-dessous
  - Si vous ne savez pas quelle révision matérielle vous avez, essayez toutes celles possibles pour votre cartouche. Seule la bonne flashrom permettra à votre linker de se lancer correctement depuis le menu HOME, mais flasher la mauvaise ne brickera pas la cartouche

#### Instructions

1. Éteignez votre console
2. Insérez votre carte SD dans votre ordinateur
   - Il s'agit de la carte SD de votre 3DS, _pas_ la carte SD de votre linker
3. Créez un dossier nommé `ntrboot` à la racine de votre carte SD
4. Copiez le fichier `.bin` venant depuis l'archive `.zip` de votre sauvegarde flashrom dans le dossier `/ntrboot/` vers la racine de votre carte SD
5. Créez un dossier appelé `payloads` dans le dossier `luma` sur votre carte SD
6. Copiez `ntrboot_flasher.firm` vers le dossier `/luma/payloads/` sur votre carte SD
7. Réinsérez votre carte SD dans votre console
8. Insérez votre linker DS/DSi compatible ntrboot dans votre console
9. Lancez ntrboot_flasher en maintenant (Start) lors du démarrage
10. Lisez l'avertissement de l'écran rouge
11. Appuyez sur (A) pour continuer
12. Sélectionnez votre linker
    - Si vous ne pouvez pas voir votre linker dans la liste en haut, lisez l'écran du bas pour en savoir plus sur chaque option
13. Sélectionnez "Restore Flash"
14. Appuyez sur (A) pour continuer
15. Attendez que le processus soit terminé
16. Appuyez sur (A) pour retourner au menu principal
17. Appuyez sur (B) pour éteindre votre console
