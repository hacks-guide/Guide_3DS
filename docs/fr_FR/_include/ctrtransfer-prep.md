1. Éteignez votre console
2. Insérez votre carte SD dans votre ordinateur
3. Créez un dossier nommé `3ds` à la racine de votre carte SD si celui-ci n'existe pas
4. Copiez `GodMode9.firm` depuis l'archive de GodMode9 `.zip` vers le dossier `/luma/payloads/` sur votre carte SD
5. Copiez le dossier `gm9` depuis l'archive GodMode9 `.zip` vers la racine de votre carte SD
6. Créez un dossier nommé `in` dans le dossier `/gm9/` si celui-ci n'existe pas déjà
7. Copiez l'image CTRTransfer `.bin` et `.bin.sha` 11.15.0 depuis l'archive CTRTransfer `.zip` vers le dossier `/gm9/in/` sur votre carte SD
8. Copiez `ctrtransfer.gm9` vers le dossier `/gm9/scripts/` sur votre carte SD
9. Copiez `FBI.3dsx` dans le dossier `/3ds/` de votre carte SD
10. Copiez `faketik.3dsx` vers le dossier `/3ds/` sur votre carte SD
11. Réinsérez votre carte SD dans votre console
