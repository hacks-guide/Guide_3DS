1. Exécuter le script MSET9 :
   - **Windows** : Double-cliquez sur `MSET9-Windows` (ou `MSET9-Windows.bat`)
   - **macOS** : Double-cliquez sur `MSET9-macOS.command` et entrez votre mot de passe si vous y êtes invité
   - **Linux** : ouvrez une fenêtre du Terminal, `cd` jusqu'à la racine de votre carte SD, puis écrivez `python3 mset9.py` et appuyez sur Entrée
