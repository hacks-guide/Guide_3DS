::: info

Si le script n'a trouvé aucun ticket utilisateur, continuez vers la section suivante.

:::

1. Lancez FBI que vous trouverez dans la liste de homebrew
2. Sélectionnez "SD"
3. Sélectionnez "gm9"
4. Sélectionnez "out"
5. Sélectionnez "ctrtransfer_tickets"
6. Effectuez la manipulation suivante pour le dossier `eshop`, ou le dossier `unknown`, ou bien les deux si vous avez les deux
   - Naviguez vers le dossier
   - Sélectionnez "\<current directory>"
   - Sélectionnez "Install and delete all tickets"
   - Attendez. Le système aura l'air de se figer, laissez-lui du temps.
   - Appuyez sur (A) pour confirmer
   - Appuyez sur (B) pour refuser l'installation des tickets depuis le CDN.
7. Appuyez sur (Home) pour quitter FBI
8. Réouvrez le Homebrew Launcher, soit via l'icône Homebrew Launcher sur le menu HOME, soit en suivant de nouveau la Section IV de cette page
9. Lancez faketik que vous trouverez dans la liste de homebrew
10. Une fois que faketik a terminé le traitement, appuyez sur (Start) pour quitter faketik
