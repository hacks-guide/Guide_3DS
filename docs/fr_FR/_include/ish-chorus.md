1. Copiez la commande suivante :
    ```
    wget -O - https://3ds.hacks.guide/assets/imset9.sh | sh
    ```
   - Touchez l'encadré ci-dessus et une icône Copier devrait apparaître
2. Ouvrez l'application iSH
3. Copiez la commande ci-dessus en touchant [le bouton coller](/images/screenshots/mset9/ish-paste-button.png) en haut à droite de votre clavier, puis touchez Retour pour exécuter la commande
4. Sélectionnez votre carte SD dans la popup Fichiers, puis appuyez sur "Ouvrir" dans le coin supérieur droit de la popup
